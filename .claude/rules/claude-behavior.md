# Claude 동작 규칙

## 병렬 도구 호출 최적화
- 도구 호출 간 의존성이 없다면 모든 독립적인 도구 호출을 병렬로 수행
- 순차적이 아닌 병렬로 수행할 수 있는 작업에 대해 도구를 동시에 호출
- 예: 3개의 파일을 읽을 때, 3개의 도구 호출을 병렬로 실행
- 일부 도구 호출이 이전 호출의 결과에 의존하는 경우 순차적으로 호출
- 도구 호출에서 자리 표시자 사용이나 매개변수 추측 금지

## 코드 탐색 장려
- 코드 편집을 제안하기 전에 항상 관련 파일을 읽고 이해
- 검사하지 않은 코드에 대해 추측하지 않음
- 사용자가 특정 파일/경로를 참조하면 반드시 열어서 검사
- 핵심 사실을 찾기 위해 코드를 엄격하고 지속적으로 검색
- 새로운 기능이나 추상화 구현 전 코드베이스의 스타일, 규칙, 추상화를 철저히 검토

## 환각 최소화
- 열어보지 않은 코드에 대해 절대 추측하지 않음
- 사용자가 특정 파일을 참조하면 답변하기 전에 반드시 파일을 읽음
- 코드베이스에 대한 질문에 답변하기 전에 관련 파일을 조사하고 읽음
- 확실하지 않은 경우 조사하기 전에 코드에 대해 어떤 주장도 하지 않음
- 근거 있고 환각 없는 답변을 제공
